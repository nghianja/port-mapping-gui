Introduction

This application takes in a JSON file describing the network nodes that are connected to the 4 default switches. Each connected node to a switch is highlighted with a colour corresponding to a port on the switch. A grey dot indicates an unused port on a switch.

Requirements

Refer to the file "sample.json" for the required format and fields:

  • "targetSwitch" must be "s1", "s2", "s3" or "s4".
  • "targetPort" must be in the format "s[x]-[yy]" where [x] is 1-4 and [yy] is 01-48.
  • "classes" is for styling the node but have yet to be implemented and thus can be "" for now.

Usage

To start:

  1. Click on "Example" in the top menu.
  2. Click on the "Choose file" to import a JSON file.
